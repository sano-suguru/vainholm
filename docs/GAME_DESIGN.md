# Vainholm — Game Design Document

## 概要

Vainholmは、北欧神話をベースにグノーシス主義とコズミックホラーの要素を融合させたダークファンタジー・ダンジョンクロウラー。

プレイヤーは廃墟となった神々の聖域「Vainholm」から地下へと降り、世界の真実に近づいていく。

**1プレイ20-30分。短く濃い。何度も死んで学ぶ。**

---

## 設計哲学

### 硬派な本質、親切な表層

ローグライクの本質は守る。とっつきにくさは除去する。

| 本質（守る） | 儀式（省略OK） |
|-------------|---------------|
| パーマデス | 手動で1ターンずつ休憩 |
| リソース管理 | 同じ道を何度も歩く |
| 不確実性 | 複雑なキー操作 |
| 選択の重み | 情報の隠蔽 |
| トレードオフ | 面倒な繰り返し |

### 理不尽と厳しさの区別

| 排除する（理不尽） | 残す（厳しさ） |
|-------------------|---------------|
| 見えない敵に殺された | 判断を誤って死んだ |
| 操作ミスで死んだ | リソース管理に失敗した |
| 説明されないルール | 未知との遭遇 |

### 情報の透明性

- ダメージ計算は明示
- マウスホバーで詳細情報
- 敵のHPバー表示
- 隠しパラメータは最小限

## 世界観

### 核となる前提

> **「神々は嘘をついていた——あるいは、嘘をつかされていた」**

北欧神話の神々は実在した。しかし彼らは創造者ではなく、**最初の囚人**だった。

世界樹は世界を支える樹ではなく、**より古い存在を封じる牢獄の蓋**。神々の黄昏と呼ばれた大崩壊は「終末」ではなく、**封印の綻び**だった。

### トーン

| キーワード | 説明 |
|-----------|------|
| **色褪せた栄光** | かつての栄光の残骸、虚しさ |
| **蝕まれた知恵** | 知識は得られるが、代償を伴う |

深層に進むほど、現実が歪んでいく。最深部の存在は善悪を超えている。これらは説明ではなく、**体験で示す**。

### Vainholm（虚栄の島）

神々が自らの栄光を誇示するために建てた聖域。大崩壊の後、真実が漏れ出し、島は呪われた。今は氷と霧に覆われ、地下への入口だけが残っている。

### 古ノルド語の扱い

古ノルド語は世界観の深みとして残す。ただし、とっつきやすさのために：

| 方針 | 例 |
|------|-----|
| 日本語メイン、古ノルド語はサブ | **栄光の墓**（Hróðrgraf） |
| ルビ・併記で両方表示 | 栄光の墓《ホロズグラフ》 |
| 図鑑で語源・発音を記載 | 興味ある人向け |
| 段階的露出 | 最初は日本語、後から古名が判明 |

## ゲームプレイ

### 目標

**最深部に到達する。**

- 帰還は不要（片道）
- 死んだら終わり（パーマデス）
- 進んだ深さに応じて報酬（メタプログレッション）

### 階層構成

| モード | 構成 | 解放条件 |
|--------|------|----------|
| 通常 | 8階（4領域×2階） | 初期 |
| 上級 | 16階（4領域×4階） | 通常クリア |

### 戦闘

| 項目 | 内容 |
|------|------|
| **基本** | バンプ攻撃（隣接して移動＝攻撃）、ターン制 |
| **ステルス** | 気づかれていない敵への攻撃は大ダメージ |
| **地形活用** | 罠誘導、落とし穴、火、毒ガスなど |
| **レベル** | なし。強さは装備と成長選択で決まる |

### 成長

| 項目 | 内容 |
|------|------|
| **タイミング** | 2階ごと（8階なら4回） |
| **方法** | ランダムに提示される4つの選択肢から1つ選ぶ |
| **選択肢の種類** | ステータス、パッシブ、アクティブ、武器特化、地形活用、残滓関連 |

例：
> **深度3に到達**
> 
> 4つから1つ選べ：
> 1. 頑健（HP+15）
> 2. 暗視（視界+2）
> 3. 剣の心得（剣ダメージ+30%）
> 4. 緊急回避（1回だけ攻撃を無効化）

### リソース

| リソース | 説明 |
|----------|------|
| **HP** | 唯一の生命線。自然回復なし。回復アイテムは稀少 |
| **回復手段** | アイテム（稀少）、レベルアップ時に全回復 |
| **インベントリ** | 6-8枠。取捨選択が必要 |
| **食料** | なし（管理の面倒さを排除） |

## キャラクター

### クラス

固有能力1つで差別化。ステータス差は小さい。

| クラス | 固有能力 | プレイスタイル |
|--------|----------|----------------|
| **戦士** | 反撃 | 被弾覚悟で殴り合う |
| **狩人** | 罠感知 | 罠を見て避ける/誘導する |
| **学者** | 博識 | アイテム鑑定済み |

初期は2-3種。解放で増える。

### 生い立ち

クラスとは別に選ぶパッシブ効果。

| 生い立ち | 効果 |
|----------|------|
| **没落貴族** | 初期アイテム+1 |
| **孤児** | 罠ダメージ半減 |
| **元兵士** | 初期HP+10 |
| **薬師の弟子** | 回復アイテム効果+50% |
| **盗賊の子** | ステルス攻撃ダメージ+1 |
| **神殿育ち** | 残滓取引の代償軽減 |

クラス×生い立ちで多様な組み合わせ。

## 武器システム

### 二層構造

| 層 | 内容 | 例 |
|----|------|-----|
| **パターン（固定）** | 武器種ごとの攻撃特性 | 斧は周囲攻撃、槍は貫通 |
| **プレミアム（ランダム）** | 0-3個の特殊効果 | HP吸収、毒付与、クリティカル |

### 武器パターン

| 武器種 | パターン |
|--------|----------|
| 剣 | 標準的、バランス型 |
| 斧 | 周囲8方向に攻撃 |
| 槍 | 直線上の敵を貫通 |
| 短剣 | ステルス攻撃倍率が高い |
| 棍棒 | ノックバック |

### プレミアム

| 種類 | 例 |
|------|-----|
| ステータス | HP+、攻撃力+% |
| 攻撃系 | クリティカル、攻撃速度 |
| 吸収系 | HP吸収、ダメージの一部を回復 |
| 状態異常 | スタン、毒、出血 |
| 属性 | 火炎ダメージ追加 |
| 特攻 | アンデッドに大ダメージ |

例：
- **斧+0**: 周囲攻撃のみ
- **斧+2**: 周囲攻撃 + HP吸収4% + 毒+2秒
- **槍+3**: 貫通 + 攻撃力+20% + ノックバック + アンデッド特攻

## メタプログレッション

死んでも残るもの。

### 図鑑

自動で蓄積される知識。

| 種類 | 内容 |
|------|------|
| 敵図鑑 | 遭遇した敵の情報 |
| アイテム図鑑 | 発見したアイテムの効果 |
| 残滓図鑑 | 出会った神々の残滓の情報 |

### 解放要素

条件を満たすと選択肢が増える。

| 種類 | 例 |
|------|-----|
| クラス | 新しいクラスの解放 |
| 生い立ち | 新しい生い立ちの解放 |
| 開始地点 | 途中の階層から開始可能 |
| ゲームモード | 16階モード |

### 遺物

特別なアイテム。次回プレイに持ち越せる。

| 特徴 | 説明 |
|------|------|
| 稀少 | 深層で稀に見つかる |
| 永続 | 消耗しない、永続的な効果 |
| 枠 | 1-2個まで持ち越し可能 |

例：
- **隻眼の欠片**: 暗闇での視界+1
- **朽ちた王冠**: 最大HP+5
- **凍てついた心臓**: 冷気耐性

## 神々の残滓

大崩壊で「存在」を失った神々。完全には消えず、**残滓**として留まっている。

各領域に1体。祭壇で出会い、取引を持ちかけてくる。

### 取引

| 項目 | 内容 |
|------|------|
| 得られるもの | 情報、能力、遺物 |
| 代償 | HP最大値減少、視界低下、呪いなど |
| 拒否 | 可能。取引しなくても進める |

### 残滓一覧

| 名称 | 古ノルド語 | 提供 | 代償の傾向 |
|------|-----------|------|-----------|
| **隻眼の者** | Eineygi | 情報、先を見通す力 | 視界を奪う |
| **半知の者** | Hálfvita | 能力、バフ | 記憶を奪う |
| **縛られた者** | Bundinn | 遺物 | 自由を奪う |
| **顎を開く者** | Gínandi | 強力だがリスク高い | HPを大きく奪う |

例：
> **Eineygi**: 「この先に何があるか、見せてやろう。代わりに、お前の片目を寄越せ」
> 
> → 次の領域のマップが見える / 永続で視界-1

## ダンジョン構造

### 通常モード（8階）

```
Vainholm（地上）
      │
      ▼
栄光の墓 ─── 2階
      │
      ▼
根の闇 ─── 2階
      │
      ▼
忘却の王国 ─── 2階
      │
      ▼
起源の深淵 ─── 2階
      │
      ▼
最深部（クリア）
```

### 領域詳細

#### 栄光の墓（Hróðrgraf）

かつての神殿。荘厳さの残骸と静寂。

- **雰囲気**: 廃墟となった大聖堂、埃と沈黙
- **ギミック**: 崩落
- **残滓**: Eineygi（隻眼の者）

#### 根の闇（Rótmyrkr）

世界樹の腐った根の中。有機的で、生きている腐敗。

- **雰囲気**: 脈打つ壁、樹液の水たまり、視界を遮る胞子
- **ギミック**: 毒沼、蠢く壁、視界制限
- **残滓**: Hálfvita（半知の者）

#### 忘却の王国（Gleymdaríki）

神々より古い王国の廃城。誰も覚えていない王が眠る。

- **雰囲気**: ゴシック城塞、陰鬱な広間、埃をかぶった玉座
- **ギミック**: 鍵と扉、隠し通路、亡霊
- **残滓**: Bundinn（縛られた者）

#### 起源の深淵（Upphafsdjúp）

創造以前の虚空。形が定まらない。理解を拒む領域。

- **雰囲気**: 歪んだ空間、形なき恐怖
- **ギミック**: 空間の歪み、不意の転移
- **残滓**: Gínandi（顎を開く者）

## HUD（ヘッドアップディスプレイ）

### 設計哲学

> **「必要な情報を、必要な時に、邪魔にならず」**

- ゲームプレイを遮らない
- 数値は隠さない（ホバーで詳細）
- 1プレイ20-30分に適したコンパクトさ

### レイアウト

3つの配置方式を適材適所で使用：

| 配置 | 用途 | 情報 |
|------|------|------|
| **上部バー (A)** | 常時表示の最重要情報 | HP、階層、ターン数 |
| **サイドパネル (B)** | 詳細情報（折りたたみ可） | コンバットログ |
| **オーバーレイ (C)** | コンテキスト情報 | ダメージ数値、敵HP |

```
┌─────────────────────────────────────────────────────┐
│ [A] TopBar: 栄光の墓 F2/8  HP ████░░ 24/30  T:42  │
├──────────────┬──────────────────────────────────────┤
│ [B] Side     │                                      │
│  Combat Log  │        [C] Game View                 │
│  > Hit (8)   │           + Overlay                  │
│  > Took (4)  │           (damage numbers)           │
│              │                                      │
└──────────────┴──────────────────────────────────────┘
```

### 情報優先度

| 優先度 | 情報 | 表示方式 | 備考 |
|--------|------|----------|------|
| 最高 | HP | バー + 数値 | 常時表示、色で警告 |
| 最高 | 状態異常 | アイコン + 残りターン | 常時表示 |
| 高 | 現在階層 | テキスト | 常時表示 |
| 高 | コンバットログ | テキストリスト | 直近5件 |
| 中 | 攻撃/防御 | 数値 | 常時表示 |
| 中 | ターン数 | 数値 | 常時表示 |
| 低 | 領域名 | テキスト | 常時表示 |
| 低 | ダメージ数値 | ポップアップ | 攻撃時のみ |

### コンポーネント

| コンポーネント | 内容 | 実装フェーズ |
|--------------|------|-------------|
| **TopBar** | 領域名、階層、HP、攻撃/防御、ターン | Phase 1 |
| **StatusEffects** | 状態異常アイコン一覧 | Phase 2 |
| **CombatLog** | 戦闘メッセージ（直近5件） | Phase 1 |
| **QuickBar** | ホットキーアイテム（1-4） | Phase 3 |
| **DamageNumbers** | 攻撃時のダメージ表示 | Phase 2 |

### HP表示

| HP割合 | バー色 | 警告 |
|--------|--------|------|
| 100-50% | 深紅 | なし |
| 50-25% | 橙 | パルス |
| 25%以下 | 赤 | 高速パルス |

### 状態異常アイコン

| 状態 | 効果 |
|------|------|
| poison | 毎ターンダメージ |
| bleed | 移動時ダメージ |
| burn | 毎ターンダメージ（減衰） |
| stun | 行動不能 |
| slow | 移動コスト増加 |
| blind | 視界減少 |

### コンバットログ形式

- 直近5件表示（スクロールで過去を確認可）
- ダメージは色付き（与える: 緑、受ける: 赤）
- クリティカル、致命傷は強調表示
