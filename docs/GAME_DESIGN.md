# Vainholm — Game Design Document

## 概要

Vainholmは、北欧神話をベースにグノーシス主義とコズミックホラーの要素を融合させたダークファンタジー・ダンジョンクロウラー。

プレイヤーは廃墟となった神々の聖域「Vainholm」から地下へと降り、世界の真実に近づいていく。

**1プレイ20-30分。短く濃い。何度も死んで学ぶ。**

---

## 設計哲学

### 三つの柱

> **「Brogueの創発的深さ」×「片道勇者の制約による緊張感」×「ハクスラの厳選快感」**

| 出典 | 取り入れる要素 | 担う役割 |
|------|---------------|---------|
| **Brogue** | 創発的システム、透明性、希少性 | 深さ |
| **片道勇者** | 時間制約、メタ進行、セッション設計 | 緊張感とリプレイ性 |
| **ハクスラ** | プレミアムシステム、厳選 | 収集欲と長期モチベーション |

### Brogueの原則

| 原則 | 本ゲームでの実装 |
|------|-----------------|
| **Less is More** | 要素を絞り、相互作用を深く |
| **Emergent Gameplay** | 属性×環境の連鎖反応 |
| **Transparency** | ダメージ計算の明示、ホバーで詳細 |
| **Meaningful Scarcity** | 強化の巻物は希少、選択に重み |
| **Informed Decision** | 「この装備を誰に？」「仲間を前に出すリスクは？」 |

### 片道勇者の原則

| 原則 | 本ゲームでの実装 |
|------|-----------------|
| **Constraint as Design** | 崩壊システムによる時間制約 |
| **Opportunity Cost** | 「留まるか進むか」「取りに戻るか諦めるか」 |
| **Visible Loss** | 崩壊に飲まれる装備・仲間が「見える」 |
| **Meta Progression** | 遺産ポイントによるクラス・装備解放 |
| **Session Design** | 崩壊が自然にゲームを終わらせる（30-45分） |
| **Ludonarrative Harmony** | 「崩壊するダンジョンからの脱出」という物語 |

### ハクスラの原則

| 原則 | 本ゲームでの実装 |
|------|-----------------|
| **Loot Hunt** | プレミアムのランダム性と厳選 |
| **Build Diversity** | プレミアムの組み合わせによるビルド |
| **Progression Feel** | 「良い装備が出た！」の快感 |
| **Long-term Goal** | ユニーク収集、メタ進行 |

### 硬派な本質、親切な表層

ローグライクの本質は守る。とっつきにくさは除去する。

| 本質（守る） | 儀式（省略OK） |
|-------------|---------------|
| パーマデス | 手動で1ターンずつ休憩 |
| リソース管理 | 同じ道を何度も歩く |
| 不確実性 | 複雑なキー操作 |
| 選択の重み | 情報の隠蔽 |
| トレードオフ | 面倒な繰り返し |

### 理不尽と厳しさの区別

| 排除する（理不尽） | 残す（厳しさ） |
|-------------------|---------------|
| 見えない敵に殺された | 判断を誤って死んだ |
| 操作ミスで死んだ | リソース管理に失敗した |
| 説明されないルール | 未知との遭遇 |

### 情報の透明性

- ダメージ計算は明示
- マウスホバーで詳細情報
- 敵のHPバー表示
- 隠しパラメータは最小限

## 世界観

### 核となる前提

> **「神々は嘘をついていた——あるいは、嘘をつかされていた」**

北欧神話の神々は実在した。しかし彼らは創造者ではなく、**最初の囚人**だった。

世界樹は世界を支える樹ではなく、**より古い存在を封じる牢獄の蓋**。神々の黄昏と呼ばれた大崩壊は「終末」ではなく、**封印の綻び**だった。

### トーン

| キーワード | 説明 |
|-----------|------|
| **色褪せた栄光** | かつての栄光の残骸、虚しさ |
| **蝕まれた知恵** | 知識は得られるが、代償を伴う |

深層に進むほど、現実が歪んでいく。最深部の存在は善悪を超えている。これらは説明ではなく、**体験で示す**。

### Vainholm（虚栄の島）

神々が自らの栄光を誇示するために建てた聖域。大崩壊の後、真実が漏れ出し、島は呪われた。今は氷と霧に覆われ、地下への入口だけが残っている。

### 古ノルド語の扱い

古ノルド語は世界観の深みとして残す。ただし、とっつきやすさのために：

| 方針 | 例 |
|------|-----|
| 日本語メイン、古ノルド語はサブ | **栄光の墓**（Hróðrgraf） |
| ルビ・併記で両方表示 | 栄光の墓《ホロズグラフ》 |
| 図鑑で語源・発音を記載 | 興味ある人向け |
| 段階的露出 | 最初は日本語、後から古名が判明 |

## ゲームプレイ

### 目標

**最深部に到達する。**

- 帰還は不要（片道）
- 死んだら終わり（パーマデス）
- 進んだ深さに応じて報酬（メタプログレッション）

### 階層構成

| モード | 構成 | 解放条件 |
|--------|------|----------|
| 通常 | 8階（4領域×2階） | 初期 |
| 上級 | 16階（4領域×4階） | 通常クリア |

### 崩壊システム（時間制約）

片道勇者の「闘」を本ゲームに適用。**崩壊するダンジョン**という形で時間制約を実現。

| 設定 | 値 | 効果 |
|------|-----|------|
| 崩壊開始 | 200ターン後 | 序盤は自由探索 |
| 崩壊速度 | 50ターン/1F | 中盤から圧力 |
| 目標プレイ時間 | 30-45分 | リプレイしやすい |

#### 崩壊の仕様

- ダンジョンは一定ターン経過で崩壊が始まる
- 崩壊は**最上層から順に**進行
- 崩壊したフロアには**戻れない**
- 崩壊に巻き込まれれば**ゲームオーバー**

#### 崩壊がもたらすジレンマ

| 状況 | トレードオフ |
|------|-------------|
| フロア探索 | 報酬 vs 崩壊リスク |
| 戦闘継続 | 経験値 vs 時間消費 |
| アイテム厳選 | 吟味 vs 崩壊接近 |
| 仲間救出 | 戦力補充 vs 時間コスト |

#### オプション：崩壊なしモード

- カジュアルモードとして崩壊なしを選択可能
- ただしスコアやランキングには反映されない
- 「制約あり」の価値を相対的に高める

### 戦闘

| 項目 | 内容 |
|------|------|
| **基本** | バンプ攻撃（隣接して移動＝攻撃）、ターン制 |
| **ステルス** | 気づかれていない敵への攻撃は大ダメージ |
| **地形活用** | 罠誘導、落とし穴、火、毒ガスなど |
| **レベル** | なし。強さは装備と成長選択で決まる |

### 創発的環境相互作用（Brogue式）

少数のシンプルなルールの**相互作用**から無限のシチュエーションを生み出す。

| 基本ルール | 連鎖可能性 |
|-----------|-----------|
| 火は草を燃やす | 煙で視界を遮る |
| 燃える草は煙を出す | 敵の遠距離攻撃を防ぐ |
| ガスは火で爆発する | 誘爆トラップ |
| 水は火を消す | 炎属性敵への対策 |
| 沼は移動を遅くする | 逃走阻害 |

#### 創発例

> 草原で敵に囲まれた！
> → 火炎の武器で草を燃やす
> → 煙で視界が遮られ、敵の遠距離攻撃を防ぐ
> → 煙に紛れて逃走
> → だが追ってきた敵の持つ松明が、逃げ込んだ部屋のガスに引火して…

### 成長

| 項目 | 内容 |
|------|------|
| **タイミング** | 2階ごと（8階なら4回） |
| **方法** | ランダムに提示される4つの選択肢から1つ選ぶ |
| **選択肢の種類** | ステータス、パッシブ、アクティブ、武器特化、地形活用、残滓関連 |

例：
> **深度3に到達**
> 
> 4つから1つ選べ：
> 1. 頑健（HP+15）
> 2. 暗視（視界+2）
> 3. 剣の心得（剣ダメージ+30%）
> 4. 緊急回避（1回だけ攻撃を無効化）

### リソース

| リソース | 説明 |
|----------|------|
| **HP** | 唯一の生命線。自然回復なし。回復アイテムは稀少 |
| **回復手段** | アイテム（稀少）、レベルアップ時に全回復 |
| **インベントリ** | 6-8枠。取捨選択が必要 |
| **食料** | なし（管理の面倒さを排除） |

## キャラクター

### クラス

固有能力1つで差別化。ステータス差は小さい。

| クラス | 固有能力 | プレイスタイル |
|--------|----------|----------------|
| **戦士** | 反撃 | 被弾覚悟で殴り合う |
| **狩人** | 罠感知 | 罠を見て避ける/誘導する |
| **学者** | 博識 | アイテム鑑定済み |

初期は2-3種。解放で増える。

### 生い立ち

クラスとは別に選ぶパッシブ効果。

| 生い立ち | 効果 |
|----------|------|
| **没落貴族** | 初期アイテム+1 |
| **孤児** | 罠ダメージ半減 |
| **元兵士** | 初期HP+10 |
| **薬師の弟子** | 回復アイテム効果+50% |
| **盗賊の子** | ステルス攻撃ダメージ+1 |
| **神殿育ち** | 残滓取引の代償軽減 |

クラス×生い立ちで多様な組み合わせ。

## パーティシステム

### 基本設計

| 項目 | 設定 | 理由 |
|------|------|------|
| 最大人数 | **主人公 + 3人（計4人）** | Brogueの緊張感を維持しつつ連携の面白さを |
| 仲間の入手 | 魔法で敵を味方化 + 救出イベント | 選択と偶然のバランス |
| 仲間の死 | **永久ロスト**（装備も消失） | 重みのある判断を生む |
| 操作 | 主人公のみ直接操作、仲間はAI追従 | 複雑さを抑える |

### 仲間のAI行動モード

| モード | 動作 | 用途 |
|--------|------|------|
| **突撃** | 敵に向かう | 前衛アタッカー |
| **追従** | 主人公の後ろに付く | 後衛・サポート |
| **待機** | その場に留まる | 囮・トラップ戦術 |

### 陣形の戦略性

狭い通路での戦闘：

```
[主人公]
[仲間A：盾役]
[仲間B：遠距離]
[仲間C：回復]
```

→ 順番に敵と接触するため、「誰を前に出すか」が生死を分ける

### 崩壊システムとの連携

- 仲間が崩壊フロアに取り残されそう
- 助けに戻る → 自分も危険
- 見捨てる → 仲間と装備をロスト

**片道勇者的体験**: 「仲間を置いて逃げるか、助けるために危険を冒すか」という道徳的・戦略的判断

## 武器システム

### 二層構造

| 層 | 内容 | 例 |
|----|------|-----|
| **パターン（固定）** | 武器種ごとの攻撃特性 | 斧は周囲攻撃、槍は貫通 |
| **プレミアム（ランダム）** | 0-3個の特殊効果 | HP吸収、毒付与、クリティカル |

### 武器パターン

| 武器種 | パターン |
|--------|----------|
| 剣 | 標準的、バランス型 |
| 斧 | 周囲8方向に攻撃 |
| 槍 | 直線上の敵を貫通 |
| 短剣 | ステルス攻撃倍率が高い |
| 棍棒 | ノックバック |

### プレミアム二層構造（青/緑）

Brogueの「透明性」を守るため、**プレミアムは拾った瞬間に全効果が分かる**（鑑定不要）。

| 色 | 名称 | 効果条件 | 設計意図 |
|----|------|----------|----------|
| **青** | 装備効果 | 装備時のみ発動 | 「誰に持たせるか」の判断 |
| **緑** | 所持効果 | 持っているだけで発動 | パーティ全体の底上げ、装備できない武器にも価値 |

### ドロップ時の視覚的識別

| 光の色 | 意味 |
|--------|------|
| 白 | プレミアムなし |
| 青 | 青プレミアムあり |
| 緑 | 緑プレミアムあり |
| 青緑 | 両方あり |
| 金 | ユニーク装備 |

### 武器プレミアム（青）

| プレミアム | 効果 | Brogue的シナジー |
|-----------|------|-----------------|
| **かず+1〜3** | 攻撃回数増加 | 毒武器なら毒スタックが加速 |
| **ダメージ+10〜50%** | 最終ダメージ乗算 | シンプルな火力 |
| **貫通** | 攻撃が敵を貫通 | 狭い通路での殲滅力 |
| **ノックバック+1〜3** | 敵を吹き飛ばす | 溶岩・穴への落下コンボ |
| **火炎付与** | 攻撃に炎追加 | 草を燃やす、ガス引火 |
| **冷気付与** | 攻撃に冷気追加 | 水面凍結、敵の減速 |
| **電撃付与** | 攻撃に電気追加 | 水中の敵に連鎖ダメージ |
| **吸命+5〜20%** | 与ダメの一部回復 | 継戦能力 |

### 防具プレミアム（青）

| プレミアム | 効果 | Brogue的シナジー |
|-----------|------|-----------------|
| **物理耐性+5〜30%** | 被ダメ軽減 | 前線維持 |
| **炎耐性/冷耐性/電耐性** | 属性軽減 | 自分の火炎戦術に巻き込まれてもOK |
| **反射+5〜15%** | 近接ダメージ反射 | 盾役の削り |
| **棘+固定値** | 攻撃してきた敵にダメージ | 待ち戦術 |

### 所持効果プレミアム（緑）

| プレミアム | 効果 | 設計意図 |
|-----------|------|----------|
| **HP+10〜50%** | 最大HP増加 | 誰でも恩恵を受けられる |
| **視界+1〜2** | 暗闘での視界拡大 | 探索の安全性 |
| **空腹耐性+10〜30%** | 食料消費減 | 長期戦向け |
| **罠感知** | 近くの罠が見える | 事故防止 |
| **忍び足** | 敵に気づかれにくい | ステルスプレイ |

**緑プレミアムのポイント**: 「この武器は自分には使えないけど、HP+30%が付いてるから持っておこう」という判断が生まれる。

### 厳選の深さ

| 品質 | 例 |
|------|-----|
| 凡庸 | ロングソード / ダメージ+12% |
| 良品 | ロングソード / かず+1 / ダメージ+25% |
| 秀逸 | ロングソード / かず+2 / 火炎付与 / HP+20%（緑） |
| 神器級 | ロングソード / かず+3 / ダメージ+48% / 吸命+18% |

### 強化の巻物

Brogueの希少リソース。ゲーム全体で**約12〜15枚**しか手に入らない。

| 使用用途 | 効果 |
|----------|------|
| 武器/防具に使用 | 基礎性能向上 |
| プレミアム枠追加 | **ランダムにプレミアム1つ追加** |

**ジレンマ**: 「今ある装備を育てるか、もっと良いのを探すか」

## ユニーク装備

### 設計思想

- **プレミアム固定**: 何が付いているか事前に（噂などで）知りうる
- **入手方法が限定的**: 特定ボス、特定イベント、超低確率ドロップ
- **強力だが万能ではない**: 状況を選ぶものもある、デメリット付き

### ユニーク装備例

| 名前 | ベース | 固定プレミアム | 特徴 |
|------|--------|---------------|------|
| **灰燼の剣** | ロングソード | 火炎付与、かず+2、自己炎上（デメリット） | 自分も燃える。炎耐性必須 |
| **凍土の斧** | ウォーアックス | 冷気付与、ノックバック+3、移動-1 | 敵を吹き飛ばし凍らせるが鈍足 |
| **貪食者の短剣** | ダガー | 吸命+35%、かず+3、空腹加速（デメリット） | 回復力抜群だが食料が倍速で減る |
| **亡者の盾** | カイトシールド | 物理耐性+40%、HP-30%（デメリット） | 硬いがHPが下がる |
| **千里眼のローブ** | レザーアーマー | 視界+5、罠感知、防御力-50% | 索敵最強だが紙装甲 |
| **王の遺産** | プレートメイル | 全耐性+20%、HP+50%、呪い（外せない） | 最強防具だが一生着続ける覚悟 |

### 入手方法

| 方法 | 例 |
|------|-----|
| **固定ドロップ** | 深層20Fの「炎の守護者」→「灰燼の剣」 |
| **イベント** | 仲間を一度も死なせずに15F到達 →「王の遺産」の隠し部屋 |
| **超低確率** | ゴブリンが0.1%で「貪食者の短剣」 |

## メタプログレッション

死んでも残るもの。

### 遺産ポイント

ランの成果に応じてポイント獲得。死亡時でも獲得可能。

| 獲得条件 | ポイント目安 |
|----------|-------------|
| 到達深度 | 1F = 10pt |
| 敵撃破数 | 10体 = 5pt |
| ボス撃破 | 1体 = 50pt |
| クリア | +200pt |

### 遺産ポイントの使用用途

| 用途 | 効果 |
|------|------|
| 新クラス解放 | 選択可能なクラスが増える |
| 初期装備の改善 | 開始時の装備選択肢が増える |
| 倉庫の拡張 | 持ち越し枠+1（最大3） |
| 特殊モード解放 | チャレンジモードなど |

### Brogue希少性との両立

| 制限ルール | 理由 |
|-----------|------|
| 持ち越せるのは「解放権」であり「アイテム実物」ではない | 各ランは常にゼロスタート |
| メタ進行は「選択肢の拡大」であり「数値的有利」ではない | Brogueの公平性維持 |

**例**:
- ✗「前回拾った+3の剣を持ち越し」
- ○「騎士クラスが解放され、選択可能に」
- ○「初期装備に"短剣"が追加され、選べるように」

### 図鑑

自動で蓄積される知識。

| 種類 | 内容 |
|------|------|
| 敵図鑑 | 遭遇した敵の情報 |
| アイテム図鑑 | 発見したアイテムの効果 |
| 残滓図鑑 | 出会った神々の残滓の情報 |

### 解放要素

条件を満たすと選択肢が増える。

| 種類 | 例 |
|------|-----|
| クラス | 新しいクラスの解放 |
| 生い立ち | 新しい生い立ちの解放 |
| 開始地点 | 途中の階層から開始可能 |
| ゲームモード | 16階モード |

### 遺物

特別なアイテム。次回プレイに持ち越せる。

| 特徴 | 説明 |
|------|------|
| 稀少 | 深層で稀に見つかる |
| 永続 | 消耗しない、永続的な効果 |
| 枠 | 1-2個まで持ち越し可能 |

例：
- **隻眼の欠片**: 暗闇での視界+1
- **朽ちた王冠**: 最大HP+5
- **凍てついた心臓**: 冷気耐性

## 神々の残滓

大崩壊で「存在」を失った神々。完全には消えず、**残滓**として留まっている。

各領域に1体。祭壇で出会い、取引を持ちかけてくる。

### 取引

| 項目 | 内容 |
|------|------|
| 得られるもの | 情報、能力、遺物 |
| 代償 | HP最大値減少、視界低下、呪いなど |
| 拒否 | 可能。取引しなくても進める |

### 残滓一覧

| 名称 | 古ノルド語 | 提供 | 代償の傾向 |
|------|-----------|------|-----------|
| **隻眼の者** | Eineygi | 情報、先を見通す力 | 視界を奪う |
| **半知の者** | Hálfvita | 能力、バフ | 記憶を奪う |
| **縛られた者** | Bundinn | 遺物 | 自由を奪う |
| **顎を開く者** | Gínandi | 強力だがリスク高い | HPを大きく奪う |

例：
> **Eineygi**: 「この先に何があるか、見せてやろう。代わりに、お前の片目を寄越せ」
> 
> → 次の領域のマップが見える / 永続で視界-1

### 通常モード（8階）

```
      ▼

      │
      ▼

      │
      ▼

```

### 領域詳細

#### 栄光の墓（Hróðrgraf）

かつての神殿。荘厳さの残骸と静寂。

- **雰囲気**: 廃墟となった大聖堂、埃と沈黙
- **ギミック**: 崩落
- **残滓**: Eineygi（隻眼の者）

#### 根の闇（Rótmyrkr）

世界樹の腐った根の中。有機的で、生きている腐敗。

- **雰囲気**: 脈打つ壁、樹液の水たまり、視界を遮る胞子
- **ギミック**: 毒沼、蠢く壁、視界制限
- **残滓**: Hálfvita（半知の者）

#### 忘却の王国（Gleymdaríki）

神々より古い王国の廃城。誰も覚えていない王が眠る。

- **雰囲気**: ゴシック城塞、陰鬱な広間、埃をかぶった玉座
- **ギミック**: 鍵と扉、隠し通路、亡霊
- **残滓**: Bundinn（縛られた者）

#### 起源の深淵（Upphafsdjúp）

創造以前の虚空。形が定まらない。理解を拒む領域。

- **雰囲気**: 歪んだ空間、形なき恐怖
- **ギミック**: 空間の歪み、不意の転移
- **残滓**: Gínandi（顎を開く者）

## HUD（ヘッドアップディスプレイ）

### 設計哲学

> **「必要な情報を、必要な時に、邪魔にならず」**

- ゲームプレイを遮らない
- 数値は隠さない（ホバーで詳細）
- 1プレイ20-30分に適したコンパクトさ

### レイアウト

3つの配置方式を適材適所で使用：

| 配置 | 用途 | 情報 |
|------|------|------|
| **上部バー (A)** | 常時表示の最重要情報 | HP、階層、ターン数 |
| **サイドパネル (B)** | 詳細情報（折りたたみ可） | コンバットログ |
| **オーバーレイ (C)** | コンテキスト情報 | ダメージ数値、敵HP |

```
┌─────────────────────────────────────────────────────┐
│ [A] TopBar: 栄光の墓 F2/8  HP ████░░ 24/30  T:42  │
├──────────────┬──────────────────────────────────────┤
│ [B] Side     │                                      │
│  Combat Log  │        [C] Game View                 │
│  > Hit (8)   │           + Overlay                  │
│  > Took (4)  │           (damage numbers)           │
│              │                                      │
└──────────────┴──────────────────────────────────────┘
```

### 情報優先度

| 優先度 | 情報 | 表示方式 | 備考 |
|--------|------|----------|------|
| 最高 | HP | バー + 数値 | 常時表示、色で警告 |
| 最高 | 状態異常 | アイコン + 残りターン | 常時表示 |
| 高 | 現在階層 | テキスト | 常時表示 |
| 高 | コンバットログ | テキストリスト | 直近5件 |
| 中 | 攻撃/防御 | 数値 | 常時表示 |
| 中 | ターン数 | 数値 | 常時表示 |
| 低 | 領域名 | テキスト | 常時表示 |
| 低 | ダメージ数値 | ポップアップ | 攻撃時のみ |

### コンポーネント

| コンポーネント | 内容 | 実装フェーズ |
|--------------|------|-------------|
| **TopBar** | 領域名、階層、HP、攻撃/防御、ターン | Phase 1 |
| **StatusEffects** | 状態異常アイコン一覧 | Phase 2 |
| **CombatLog** | 戦闘メッセージ（直近5件） | Phase 1 |
| **QuickBar** | ホットキーアイテム（1-4） | Phase 3 |
| **DamageNumbers** | 攻撃時のダメージ表示 | Phase 2 |

### HP表示

| HP割合 | バー色 | 警告 |
|--------|--------|------|
| 100-50% | 深紅 | なし |
| 50-25% | 橙 | パルス |
| 25%以下 | 赤 | 高速パルス |

### 状態異常アイコン

| 状態 | 効果 |
|------|------|
| poison | 毎ターンダメージ |
| bleed | 移動時ダメージ |
| burn | 毎ターンダメージ（減衰） |
| stun | 行動不能 |
| slow | 移動コスト増加 |
| blind | 視界減少 |

### コンバットログ形式

- 直近5件表示（スクロールで過去を確認可）
- ダメージは色付き（与える: 緑、受ける: 赤）
- クリティカル、致命傷は強調表示
